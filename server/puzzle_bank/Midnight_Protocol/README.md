# Midnight Protocol

**Objective**: Decrypt the high-security transmission string.

We've intercepted a transmission from the dark side of the station. It appears to be encrypted using a simple XOR cyclic shift.

**The Hex Code**: `0x1A1B1C`
**The Key**: `0x0F`

**The Task**:
Perform a bitwise XOR operation on each of the three bytes using the key provided. 

Result Format: A 6-character hex string (e.g., `ABCDEF`).

*Note: 1A XOR 0F = ?*
